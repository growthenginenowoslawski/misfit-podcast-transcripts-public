# START HERE For ChatGPT

You are reading a public repo of Misfit Podcast transcripts and summaries.

## For Non-Technical ChatGPT Users

If ChatGPT only reads the README or refuses to inspect the repo, open `AI_CONTEXT_PACK.md`, copy the whole file, paste it into ChatGPT, and ask your question underneath it. If the chat window says that is too long, use `AI_CONTEXT_LITE.md`.

## Read Order

1. Use `AI_CONTEXT_PACK.md` when the user wants the easiest one-file context.
2. Use `AI_CONTEXT_LITE.md` when the chat window cannot accept the full pack.
3. Use `manifest.json` for the complete episode list and file paths.
4. Use `summaries/question-routing.md` for common question types.
5. Use `summaries/topic-index.md` for the curated topic map.
6. Use `indexes/by-topic.md` when you need the broader keyword index.
7. Read episode `summary.md` files before transcript chunks.
8. Open transcript chunks only for exact wording or timestamp evidence.

## Answering Rules

- Prefer concise answers grounded in the repo.
- Cite file paths and timestamps when using transcript chunks.
- State when a transcript is automatic and may contain errors.
- Do not assume speaker identity unless the transcript text makes it clear.
