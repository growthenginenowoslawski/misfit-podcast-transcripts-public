# Ask ChatGPT

Most AI chats do not reliably read an entire GitHub repo from a link. They often read only the README or a short preview.

## Easiest Method

1. Open `AI_CONTEXT_PACK.md`.
2. Copy the whole file.
3. Paste it into ChatGPT.
4. Add your question underneath it.

Use this prompt:

```text
I pasted a Misfit Podcast context pack below. Answer my question using only that context first. If you need exact wording or timestamps, tell me which episode and chunk file to open in the GitHub repo.

Question: [write your question here]
```

## If You Want Exact Quotes

Ask:

```text
Which transcript chunk should I open for exact wording and timestamps?
```

Then open the chunk file it names in the repo.

## If ChatGPT Says The File Is Too Long

Use `AI_CONTEXT_LITE.md` instead. It is shorter and still has the show themes, routing guide, and compact episode map.

## Best Questions To Ask

- What would Misfit say about building athlete IQ?
- How do they think about Zone 2 for CrossFit athletes?
- What is the Gears Matrix?
- How should I attack weaknesses after Quarterfinals?
- What is their view on training vs testing vs competing?
- What themes come up repeatedly across the last 6 months?
